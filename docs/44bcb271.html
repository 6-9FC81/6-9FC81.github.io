第一次ssh连接

  • 输入 ssh [远程用户名]@[IP地址] -A 开始连接，之后输入密码即连接成功

配置密钥

  • 生成密钥对：在cmd窗口中输入命令 ssh-keygen ，生成的密钥保存在 C:/Users/[本地用户名]/.ssh/

  • 公钥：C:/Users/[本地用户名]/.ssh/id_rsa.pub 上传到linux内：

    • 如果 [远程用户名] 是root，则该文件传输到 ./.ssh 目录下(根目录下的.ssh文件夹)
    • 如果 [远程用户名] 非root，则该文件传输到 ~/.ssh 目录下(要连接的用户的用户文件夹的根目录下的.ssh文件夹)
    • cd 到该.ssh文件夹，输入命令 cat id_rsa.pub >> authorized_keys
