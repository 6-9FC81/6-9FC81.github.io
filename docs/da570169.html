结构体

tag 是结构体标签，用于下次定义一个另一个（相同结构的）结构体变量。

member-list 是标准的变量定义，比如 int i; 或者 float f，或者其他有效的变量定义。

variable-list 结构变量，是定义该结构体时，初始定义的结构体变量，相当于定义一个变量时立即初始化赋值。就像赋值与否都不影响变量的定义，有没有结构变量都不影响 struct tag 的定义。

注：定义结构体变量时必须使用 struct tag 定义。
