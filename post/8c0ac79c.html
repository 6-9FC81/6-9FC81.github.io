  • 下载 git 安装包、安装步骤详见：[Windows系统Git安装教程（详解Git安装过程）](Windows系统Git安装教程（详解Git安装过程） - 学为所用 - 博客园 (cnblogs.com))

  • 注册GitHub账号步骤详见：自行百度教程。

  • 生成 SSH key

    • 生成SSH key

      • 回到命令提示符(管理员)，将命令中的your_email@email.com换成自己注册GitHub的邮箱，运行。

      • 生成过程中会多次要求确认，不停按回车即可

    • 查看生成的SSH key

      • 打开 C:\Users\Usersname\.ssh\ 其中，Usersname是你自己电脑的登录用户名。
      • 其中，id_rsa 是私钥，id_rsa.pub 是公钥。公钥上传给服务器，私钥用来与公钥配对。
